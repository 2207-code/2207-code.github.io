s/help raid - отобразить список команд данной категории
s/spamhook [текст] - Создаёт вебхук с текущем канале, и спамит текстом по вебхуку
s/spamhere [число] [текст] - Отправляет текст в текущий канал укзанное количество раз, если число не указано, будет спамить 100 раз
s/randspamhere [число] [текст] - Отправляет текст в текущий канал укзанное количество раз, добавляя к тексту рандомные строки, если число не указано, будет спамить 100 раз
s/raid [текст] - Спамит указанным текстом во все текстове каналы сервера
s/randraid [текст] - Спамит указанным текстом во все текстове каналы сервера, добавляя к тексту рандомные строки

s/help perms - отобразить список команд данной категории
s/userperms [пользователь] - Отображает серверные права указанного пользователя. В качестве пользователя может выступать: айди, упоминание, ник, ник и тег
s/roleperms [роль] - Отображает серверные права указанной роли. В качестве роли может выступать: айди, упоминание, имя роли
s/adminfinder - Приводит список ролей сервера с админкой и их айди

s/help chat - отобразить список команд данной категории
s/embtext [текст] - Отображает серверные права указанного пользователя. В качестве пользователя может выступать: айди, упоминание, ник, ник и тег
s/coloremb [цвет] [текст] - Аналог s/embtext, но позволяет установить цвет. Список цветов можно найти ниже

s/help crash - отобразить список команд данной категории
s/grantadmin [пользователь] - Выдаёт админ-права указанному пользователю. В качестве пользователя может выступать: айди, упоминание, ник, ник и тег
s/crchannels - Массовое создание каналов на сервере
s/croles - Массовое создание ролей на сервере
s/delroles - Удаление всех возможных ролей сервера
s/delchannels - Удаление всех возможных каналов сервера

s/help status - отобразить список команд данной категории
s/gamestatus [текст] - Установит статус "Играет в (текст)". Если текст не указан, он будет "http://matvey2207.gq/selfbot"
s/musicstatus [текст] - Установит статус "Слушает (текст)". Если текст не указан, он будет "http://matvey2207.gq/selfbot"
s/streamstatus [текст] - Установит статус "Стримит (текст)". Если текст не указан, он будет "http://matvey2207.gq/selfbot"
s/watchingstatus [текст] - Установит статус "Смотрит (текст)". Если текст не указан, он будет "http://matvey2207.gq/selfbot"
s/competingstatus [текст] - Установит статус "Соревнуется в в (текст)". Если текст не указан, он будет "http://matvey2207.gq/selfbot" Примечание: в старых клиентах дискорд отображается как "Играет"
